# Port Scanner

Bu Python programı, belirtilen web sitesinin açık portlarını kontrol etmek için kullanılır.

## Nasıl Kullanılır

1. Programı çalıştırmak için Python 3 gereklidir.
2. `portscanner.py` dosyasını indirin veya kopyalayın.
3. İndirdiğiniz dizinde ki betiği çalıştırın.
4. Program çalıştırıldığında, hedef web sitesinin hostname'ini ve kontrol edilecek portları girin.
5. Program, her bir port için bağlantı deneyimi yapacak ve açık portları ekrana yazdıracaktır.

## Dikkat Edilmesi Gerekenler
Port taraması, yasal ve etik kurallara uygun olarak yapılmalıdır.
Başkalarının sistemlerine izinsiz olarak port taraması yapmak yasa dışıdır ve etik olmayabilir.
Programı yalnızca izinli sistemlerde kullanın ve gerekli izinleri alın.
